YCSB Commands to setup:

curl -O --location https://github.com/brianfrankcooper/YCSB/releases/download/0.14.0/ycsb-0.14.0.tar.gz

tar xfvz ycsb-0.14.0.tar.gz

cd ycsb-0.14.0



Run YCSB:

./bin/ycsb



YCSB Commands for executions:

$ ./bin/ycsb load basic -P workloads/workloadb

$ ./bin/ycsb run basic -P workloads/workloadb > log.txt
