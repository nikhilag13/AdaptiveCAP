After Clustering

What happens before clustering?

Size bounded cluster

This cluster is formed when each node in the intial spanning tree (graph) sends size message to its parent. Parent checks if the sum of its size and the node size is greater than a threshold. If it is greater, prune message is sent and node that sent the message becomes a new cluster head. If not, node gets added to the existing cluster. At the end of this, each node belongs to a cluster by adhering to the size constraint.

What happens after clustering?

Reduction in Energy and Latency

After intial clustering, each node sends hello messages to other neighboring nodes and based on the distance from its clusterhead, nodes are moved into different clusters to reduce latency.
